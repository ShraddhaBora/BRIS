# Contributing to BRIS

Thank you for your interest in contributing to the Behavioral Risk Intelligence System (BRIS). As a research-focused project, we prioritize contributions that enhance the predictive validity, interpretability, and robustness of the system.

## Research Areas of Interest
We welcome contributions in the following areas:
- **Feature Engineering**: Innovative behavioral markers derived from open banking or transaction sequences.
- **Model Calibration**: Improvements to Platt scaling or alternative calibration techniques (Isotonic Regression, Temperature Scaling).
- **Explainability**: Implementation of alternative attribution methods (IME, Integrated Gradients).
- **Aesthetic Refinement**: Enhancing the clarity and precision of the analytical interface.

## Submission Guidelines
1. Fork the repository and create a new branch for your feature or research finding.
2. Ensure any new model logic includes a corresponding explanation in the project documentation.
3. Maintain the consistent code style established in the existing React and Python components.
4. Submit a Pull Request with a detailed summary of the behavioral or technical justification for your changes.

## Code of Conduct
We are committed to fostering a collaborative and professional research community. All contributors are expected to uphold high standards of academic and technical integrity.
