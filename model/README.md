# BRIS Model Backend

Real ML pipeline for the Behavioral Risk Intelligence System.

## Quick Start

```bash
cd BRIS/model
pip install -r requirements.txt

# Train models (generates synthetic data by default, ~2-3 min)
python train.py

# Start API server
uvicorn api:app --reload --host 0.0.0.0 --port 8000
```

The frontend at `http://localhost:5173` will automatically detect the API
and switch from the formula fallback to real model predictions.
Watch for the **"Real model"** badge in the analyzer results panel.

---

## Using the Real Kaggle Dataset

1. Download **cs-training.csv** from https://www.kaggle.com/c/GiveMeSomeCredit/data
2. Place it at `model/data/cs-training.csv`
3. In `train.py`, set `USE_REAL_DATA = True`
4. Re-run `python train.py`

## API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| `GET`  | `/health`  | Server status and model load state |
| `POST` | `/predict` | Run prediction on a subject |
| `GET`  | `/metrics` | Evaluation metrics from training run |

### POST /predict — Request body

```json
{
  "credit_utilization":  0.75,
  "payment_delay":       30,
  "spending_volatility": 0.48,
  "income_stability":    0.70,
  "missed_payments":     1,
  "avg_balance":         12000,
  "monthly_income":      5000,
  "age":                 42,
  "employment_status":   "employed"
}
```


## What the Metrics Mean

- **AUC-ROC**: Probability a random defaulter scores higher than a random non-defaulter. `> 0.80` is industry standard.
- **KS Statistic**: Maximum separation between the default and non-default score distributions. `> 0.40` is strong.
- **Gini Coefficient**: `= 2 × AUC − 1`. Measures rank-ordering power.
- **Brier Score**: Mean squared error of probability estimates. Lower is better.
- **AUC-PR**: More informative than AUC-ROC for imbalanced datasets (~6.7% positive rate).
